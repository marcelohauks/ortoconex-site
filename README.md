# Ortoconex Site

Base inicial do novo site da OrtoConex usando a mesma stack do projeto Gloria:

- Next.js (App Router)
- React + TypeScript
- Tailwind CSS

## Status

- Estrutura limpa criada
- Home temporária: "Site em construção"

## Executar localmente

Pré-requisitos:

- Node.js 20+
- npm 10+

Instale as dependências:

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).
